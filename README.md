# Venus Proof Verify Demo

This demo verifies a Venus `vadcop_final_proof.bin` proof by:

1. Loading the wrapped proof file generated by `cargo-zisk prove`. The proofs can be downloaded from [cysic cluster v3](https://ethproofs.org/clusters/2d1e3c1b-5f53-418f-bae1-6952ffae35ff).
2. Reconstructing the inner `VadcopFinalProof`
3. Packing it into the raw `proof_with_publics()` format expected by the low-level verifier
4. Calling [verify_vadcop_final_bytes()](https://github.com/ethproofs/venus/blob/124384dd88e799ffe710e09b82ca124c38afb0de/verifier/src/verifier.rs#L23) or [verify_vadcop_final_compressed_bytes()](https://github.com/ethproofs/venus/blob/124384dd88e799ffe710e09b82ca124c38afb0de/verifier/src/verifier.rs#L21)

## Run

```bash
git clone https://github.com/cysic-labs/venus_proof_verify_demo.git
cd venus_proof_verify_demo
make download
make verify
```
