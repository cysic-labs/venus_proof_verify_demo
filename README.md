# Venus Proof Verify Demo

This demo verifies a Venus `vadcop_final_proof.bin` proof by:

1. Loading the wrapped proof file generated by `cargo-zisk prove`
2. Reconstructing the inner `VadcopFinalProof`
3. Packing it into the raw `proof_with_publics()` format expected by the low-level verifier
4. Calling `verify_vadcop_final_bytes()` or `verify_vadcop_final_compressed_bytes()`

## Requirements

- Network access to fetch crates from:
  - `https://github.com/ethproofs/venus`
- A wrapped proof file such as:
  - `../venus_v16_testbuild/tmp/vadcop_final_proof.bin`
- The matching VADCOP final verification key such as:
  - `../venus_v16_testbuild/build/provingKey/zisk/vadcop_final/vadcop_final.verkey.bin`

## Run

```bash
cd ~/eth_proof/eth_proof_sihao/venus_proof_verify_demo
cargo run -- \
  /home/cysic/eth_proof/eth_proof_sihao/venus_v16_testbuild/tmp/vadcop_final_proof.bin \
  /home/cysic/eth_proof/eth_proof_sihao/venus_v16_testbuild/build/provingKey/zisk/vadcop_final/vadcop_final.verkey.bin
```

Expected output:

```text
proof=/.../vadcop_final_proof.bin
vk=/.../vadcop_final.verkey.bin
compressed=false
verified=true
```
